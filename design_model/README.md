# Design Model

> Status: Incubating
> Alignment: Not yet aligned with the current open-standard rewrite
> Intended role: Earlier design-oriented adapter material now treated as an
> extraction source
> Use it for: selective extraction of reusable patterns, not as a promoted
> example or current builder-path artifact

This directory contains earlier design-oriented adapter material that may still
be useful, but it is not currently part of the aligned standard surface.

Use it when you want to inspect possible source material for:

- design-oriented template ideas
- design-specific verification language
- artifact and quality expectations that may justify future guide or template
  refinement

Do not treat this directory as a near-term candidate for direct promotion into
the current example set.
Promote only extracted pieces that fill a builder need not already covered by
the current curated examples.
